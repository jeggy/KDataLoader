Install using gradle

In order to include KDataLoader with gradle you need to include jcenter as repository.

repositories {
    jcenter()
}

Then you can add the dependency.

Groovy
dependencies {
    implementation 'de.nidomiro:KDataLoader:0.0.2'
}
kts
dependencies {
    implementation("de.nidomiro:KDataLoader:0.0.2")
}
